 Supermarket Billing & Inventory Management System

 Overview

A full-stack supermarket billing and inventory management system developed using Flask, MySQL, HTML, CSS, and Python.

 Features

 Product Management

* Add Product
* Edit Product
* Delete Product
* Restock Product
* Category-wise Product Filtering
* Stock Status Tracking

 Billing System

* Multi-product Billing
* Shopping Cart
* GST Calculation (5%)
* Discount Support
* Multiple Payment Methods

 Inventory Management

* Automatic Stock Reduction
* Restock History
* Low Stock Monitoring

 Reports

* Total Sales
* Total Bills
* Total Customers
* Top Selling Products
* Low Stock Products

 Bill Management

* Receipt Generation
* Bill History
* Receipt PDF Download

Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* JavaScript

 Database Tables

* sm_categories
* sm_products
* sm_customers
* sm_bills
* sm_bill_items
* sm_payments
* sm_restock_history

 Future Enhancements

* Barcode Scanning
* Sales Analytics Charts
* Email Receipt System
* User Authentication

  Update db_config.py with your own MySQL credentials before running the project.
